# Hogwarts Legacy - Game Website UI Concept

A static landing page concept for a gaming website. This project was built to showcase a modern UI/UX approach for game marketing. 

Please note: This is purely a visual UI mockup. The navigation and secondary buttons are static placeholders. The only functional interactive element is the "Watch Trailer" button, which opens a cinematic video modal.

## Live Demo

> Click here to view the live website: https://intanamandaa.github.io/hogwarts-legacy/

## Features

- Immersive Hero Section: Full-screen ambient background video to capture the magical gaming atmosphere.
- YouTube Integration: A functional "Watch Trailer" button that seamlessly triggers a YouTube video inside a sleek dark modal.
- Thematic Design: Custom typography using Cinzel and Fauna One fonts, combined with gold-glow CSS effects tailored to the Hogwarts theme.
- Responsive Layout: Optimized for mobile, tablet, and desktop screens using Tailwind CSS.

## Tech Stack

- HTML5
- Tailwind CSS (via CDN)

## About

This project was initially created as a university assignment for the Digital Game Development course. The main objective was to practice frontend layouting, specifically focusing on integrating HTML5 video backgrounds and creating interactive modal components for YouTube embeds without relying on heavy JavaScript frameworks.
